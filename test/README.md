# Test application

This program exists to display the various test TMXs in the 'assets' folder. These test TMXs make use of most features 
raytmx supports and all should be displayed whenever raytmx is modified.

## Usage

The program accepts one command-line argument, an index.  
./test 0  
./test 1  
...and so on. Refer to the source for the index-to-TMX mappings.

Arrow keys pan the camera.
Keypad + and - zoom the camera in and out, respectively.
